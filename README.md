# Recipe Book

A simple and interactive Recipe Book web application built with **HTML**, **CSS**, and **JavaScript**. Easily browse, add, and manage your favorite recipes—all in your browser!

## Features

- **Add new recipes** with title, ingredients, and instructions
- **View all recipes** in a clean, card-style layout
- **Search and filter** recipes by name or ingredient
- **Edit and delete** existing recipes
- **Responsive UI** for desktop and mobile devices
- Data persists using **Local Storage** (no backend required)

## Installation & Usage

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Phaninder12/Recipe-Book-.git
    ```
2. **Navigate to the project folder:**
    ```bash
    cd Recipe-Book-
    ```
3. **Open `index.html` in your browser:**
    - You can simply double-click `index.html` or right-click and choose "Open with" your favorite browser.
    - No server is required; everything runs in the browser.
